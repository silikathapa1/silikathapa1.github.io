Database Projects

Sales Database

Sales database was designed for IT-640 Advanced Database Management System class. The major tables in this database are product, order, vendor, customer and employee. The database is normalized to 3NF and several PL/SQL and TSQL was written for managing sales transactions like product-order, product-return, back-order, employee-commission, quantity on hand triggers and so on. To load data into this database, excel sheet was used, firstly manual loading was done and further Business Intelligence tools as SQL Server Integration Services was used for data mapping and loading, SQL Server Reporting Services was used for generating reports and SQL Server Analysis Services was used for multidimensional analysis of data.
